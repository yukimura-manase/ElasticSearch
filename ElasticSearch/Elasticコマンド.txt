
< ElasticSearchの起動 >

# ElasticSearchディレクトリに移動
    cd elasticsearch-7.0.0

# ElasticSearchのプラグイン日本語検索プラグイン「 kuromoji 」をインストール
    bin/elasticsearch-plugin install analysis-kuromoji


# Elasticsearch を起動
    bin/elasticsearch

# ローカルホスト9200番でサーバーが立てば環境構築成功！！
    http://localhost:9200/

< Kibanaの起動 >

# Kibanaディレクトリに移動
    cd kibana-7.0.0-darwin-x86_64

    bin/kibana
